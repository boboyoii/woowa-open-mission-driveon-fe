# 🚗 DriveOn

## 프로젝트 소개

DriveOn은 자동차가 연료가 떨어지기 전까지 얼마나 멀리 달릴 수 있는지에 도전하는 게임입니다.
우테코 2주차 자동차 경주 미션에서 아이디어를 얻어 기존 미션의 요구사항을 스스로 새롭게 정의하고
콘솔 환경에서 벗어나 직접 움직이고 반응하는 시각적인 게임으로 확장했습니다.

또한 단순히 게임만 동작하는 것이 아니라 플레이어의 기록을 서버와 연결해 실제 DB에 저장하고 조회하는 기능까지
프론트엔드–백엔드–배포까지 이어지는 전체 개발 흐름을 경험해보는 것을 목표로 했습니다.

## 기술 스택

- 게임 엔진 : Phaser 3
- 개발 서버 / 빌드 도구 : Vite
- 언어 : JavaScript (ES6)

## 게임 기능

#### 플레이어 닉네임 입력 규칙

- 최대 10자까지 입력 가능하다.
- 닉네임 안에 공백은 포함할 수 없다.
- 한글 / 영문 / 숫자 / `-`, `_` 만 사용 가능하다.

#### 게임 시작

- [x] 사용자는 게임 시작 전 자신의 자동차 닉네임을 입력한다.
- [x] 시작 버튼을 누르면 게임이 시작된다.

#### 자동차 이동

- [x] 플레이어는 **좌(←)**, **우(→)** 방향키로 자동차를 이동할 수 있다.
  - [x] 자동차는 도로 경계선을 벗어날 수 없다.

#### 게임 UI 표시

- [x] 게임 시작 후 사용자의 닉네임이 화면에 표시된다.
- [x] 연료 게이지 바를 통해 현재 연료량을 확인할 수 있다.
- [x] 장애물과 연료 아이템으로 인한 연료 변화량이 표시된다.
- [x] 게임 화면 오른쪽 상단에 현재 주행 거리를 표시한다.

#### 장애물 생성

- [x] 일정 시간 간격으로 도로 위에 장애물이 무작위 위치에 생성된다. - [x] 장애물은 위에서 아래로 일정 속도로 내려온다.

#### 장애물 충돌

- [x] 자동차와 장애물이 겹치면 충돌로 판정한다.

#### 연료 감소

- [x] 시간이 지남에 따라 연료가 감소한다.
- [x] 장애물 충돌 시 연료가 추가로 감소된다.

#### 연료 아이템 생성

- [x] 도로 위에 연료 아이템이 무작위 위치에 생성된다. (<img width="25" alt="Image" src="https://github.com/user-attachments/assets/f7dceaaf-0d2d-4142-beb3-b9501b84a6bc" />)
- [x] 연료 아이템은 위에서 아래로 일정 속도로 내려온다.

#### 연료 획득

- [x] 주행 중 생성되는 연료 아이템을 획득하면 연료가 증가한다.

#### 게임 종료

- [x] 연료가 0이 되면 게임이 종료되고 결과 화면으로 이동한다.

#### 게임 종료 후 UI 표시

- [x] 최종 주행 거리를 화면에 표시한다.
- [x] 메인 화면으로 이동하는 홈 버튼을 표시한다.
- [x] 게임을 다시 시작하는 재시작 버튼을 표시한다.
- [x] 랭킹 페이지로 이동하는 랭킹 버튼을 표시한다.

#### 재시작

- [x] 플레이어는 방금 입력했던 닉네임으로 바로 다시 게임을 시작한다.

#### 랭킹 기능

- [x] 게임이 끝나면 닉네임과 주행 기록을 서버로 보내 저장한다.
- [x] 사용자는 주행 거리에 따라 정렬된 상위 10명의 랭킹을 확인할 수 있다.

## 게임 진행 흐름

```
[홈 화면]
  • "RANKINGS" → 랭킹 목록 확인
  • 닉네임 입력 칸
  • "START"
닉네임을 입력 후 "START" 버튼을 누르면 게임이 시작된다
     ↓
──────────────────────────
      🕹 [게임 플레이 화면]
──────────────────────────
  • 자동차 조작 (좌/우 방향키)
  • 시간이 지나면서 연료 감소
  • 장애물 충돌 시 연료 추가 감소
  • 연료 아이템 획득 시 연료 증가
──────────────────────────
연료가 0이 되는 순간 게임 종료
     ↓
[게임 종료 화면]
  • "RESTART" → 다시 플레이
  • "HOME" → 시작 화면 이동
  • "RANKINGS" → 랭킹 목록 확인
```

## 디렉터리 구조 & 역할 요약

| 디렉터리          | 역할 설명                                         |
| ------------- | --------------------------------------------- |
| core/    | 게임 기본 설정 (상수, 이미지, 공통 UI)            |
| managers/ | 장애물, 연료 아이템, 도로 움직임 관리        |
| objects/  | 주요 객체 정의 (Player, Fuel)       |
| scenes/   | 화면 단위 구성 (Home / Play / Result / Ranking) |
| ui/       | 게임 안에서 쓰는 UI 요소들 (연료바, 거리 표시)                 |
| utils/    | 유틸 함수 (닉네임 검증)                  |
| main.js   | 게임 실행 파일                     |


## 로컬 실행 방법 

```
# 의존성 설치
npm install

# 개발 서버 실행 (기본 포트: http://localhost:5173)
npm run dev
```
