# 🚗 DriveOn

## 프로젝트 소개

DriveOn은 자동차가 연료가 떨어지기 전까지 얼마나 멀리 달릴 수 있는지에 도전하는 게임입니다.
우테코 2주차 자동차 경주 미션에서 아이디어를 얻어 기존 미션의 요구사항을 스스로 새롭게 정의하고
콘솔 환경에서 벗어나 직접 움직이고 반응하는 시각적인 게임으로 확장했습니다.

## 기술 스택

- 게임 엔진 : Phaser 3
- 개발 서버 / 빌드 도구 : Vite
- 언어 : JavaScript (ES6)

## 게임 기능

### 게임 시작

- [x] 사용자는 게임 시작 전 자신의 자동차 닉네임을 입력한다.
- [x] 시작 버튼을 누르면 게임이 시작된다.

### 자동차 이동

- [x] 플레이어는 **좌(←)**, **우(→)** 방향키로 자동차를 이동할 수 있다.
  - [x] 자동차는 도로 경계선을 벗어날 수 없다.

### 게임 UI 표시

- [x] 게임 시작 후 사용자의 닉네임이 화면에 표시된다.
- [x] 연료 게이지 바를 통해 현재 연료량을 확인할 수 있다.
- [x] 장애물과 연료 아이템으로 인한 연료 변화량이 표시된다.
- [x] 게임 화면 오른쪽 상단에 현재 주행 거리를 표시한다.

### 장애물 생성

- [x] 일정 시간 간격으로 도로 위에 장애물이 무작위 위치에 생성된다.
- [x] 장애물은 위에서 아래로 일정 속도로 내려온다.

### 장애물 충돌

- [x] 자동차와 장애물이 겹치면 충돌로 판정한다.

### 연료 감소

- [x] 시간이 지남에 따라 연료가 감소한다.
- [x] 장애물 충돌 시 연료가 추가로 감소된다.

### 연료 아이템 생성

- [x] 도로 위에 연료 아이템이 무작위 위치에 생성된다.
- [x] 연료 아이템은 위에서 아래로 일정 속도로 내려온다.

### 연료 획득

- [x] 주행 중 생성되는 연료 아이템을 획득하면 연료가 증가한다.

### 게임 종료

- [x] 연료가 0이 되면 게임이 종료되고 결과 화면으로 이동한다.

### 게임 종료 후 UI 표시

- [ ] 최종 주행 거리를 화면에 표시한다.
- [x] 메인 화면으로 이동하는 홈 버튼을 표시한다.
- [x] 게임을 다시 시작하는 재시작 버튼을 표시한다.
- [ ] Top10 랭킹 페이지로 이동하는 랭킹 버튼을 표시한다.

### 재시작

- [x] 플레이어는 방금 입력했던 닉네임으로 바로 다시 게임을 시작한다.

### 랭킹 기능

- [ ] 사용자는 게임 종료 후 자신의 닉네임과 주행 거리를 기록으로 저장할 수 있다.
- [ ] 사용자는 주행 거리를 기준으로 정렬된 랭킹 목록을 확인할 수 있다.

## 게임 진행 흐름

```
[시작 화면]
  ↓
사용자가 닉네임을 입력한다
  ↓
"게임 시작" 버튼을 누른다
  ↓
──────────────────────────
    🕹 [게임 플레이 화면]
──────────────────────────
• 자동차 조작 (좌/우)
• 시간이 지나면서 연료 감소
• 장애물 충돌 시 연료 추가 감소
• 연료 아이템 획득 시 연료 증가
──────────────────────────
  ↓
연료가 0이 되는 순간 게임 종료
  ↓
[게임 종료 화면]
  • "재시작" → 다시 플레이
  • "홈" → 시작 화면으로 이동
  • "랭킹" → 랭킹 목록 확인
```

## 디렉터리 구조

```
src/
 ├─ core/
 │   ├─ constants.js     # 전역 상수
 │   ├─ assets.js        # 이미지 등록
 │   └─ ui.js            # 공통 UI 요소
 │
 ├─ managers/
 │   ├─ RoadManager.js       # 도로 애니메이션 및 경계 관리
 │   ├─ ObstacleManager.js   # 장애물 생성, 이동, 제거 관리
 │   └─ FuelManager.js       # 연료 아이템 생성, 이동, 제거 관리
 │
 ├─ objects/
 │   ├─ Fuel.js          # 연료 상태 관리
 │   └─ Player.js        # 플레이어 상태 관리
 │
 ├─ scenes/
 │   ├─ HomeScene.js     # 홈 화면
 │   ├─ PlayScene.js     # 플레이 화면
 │   └─ ResultScene.js   # 게임 종료 화면
 │
 └─ ui/
     └─ FuelBar.js        # 연료 게이지 UI

```

## 실행 방법

```
# 의존성 설치
npm install

# 개발 서버 실행 (기본 포트: http://localhost:5173)
npm run dev
```

| 키          | 동작                  |
| ----------- | --------------------- |
| ← / →       | 자동차 좌우 이동      |
| 마우스 클릭 | 버튼(홈, 재시작) 선택 |
