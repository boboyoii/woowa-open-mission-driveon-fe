# 🚗 DriveOn

## 프로젝트 소개

DriveOn은 자동차가 연료가 떨어지기 전까지 얼마나 멀리 달릴 수 있는지에 도전하는 게임입니다.
우테코 2주차 자동차 경주 미션에서 아이디어를 얻어
콘솔 환경에서 벗어나 직접 움직이고 반응하는 시각적인 게임으로 확장했습니다.

이번 프로젝트에서는 기존 미션의 요구사항을 스스로 새롭게 정의하고
단순히 결과를 출력하는 게 아니라 눈으로 보고 조작할 수 있는 경험으로 확장하는 게 목표입니다.

## 기능

### 1단계: 기본 기능

### 닉네임 입력 및 게임 시작

- [x] 사용자는 게임 시작 전 자신의 자동차 닉네임을 입력한다.
- [x] 닉네임을 입력하고 시작 버튼을 누르면 게임이 시작된다.
- [x] 닉네임은 화면 상단 왼쪽에 표시된다.

### 자동차 이동

- [x] 플레이어는 **좌(←)**, **우(→)** 방향키로 자동차를 이동할 수 있다.
- [x] 자동차는 도로 경계선을 벗어날 수 없다.

### 장애물 생성

- [x] 일정 시간 간격으로 도로 위에 장애물이 무작위 위치에 생성된다.
  - 장애물의 위치는 난수로 결정된다.
- [x] 장애물은 위에서 아래로 일정 속도로 내려온다.

### 충돌 판정

- [x] 자동차와 장애물의 위치가 겹치면 충돌로 간주한다.
- [x] 충돌 시 게임은 즉시 종료된다.
  - 게임 결과 화면으로 이동하고 "Game Over"를 표시한다.

### 게임 종료 이후

- [x] 홈 버튼을 누르면 메인 화면으로 이동한다.
- [x] 재시작 버튼을 누르면 게임이 다시 시작된다.

## 시스템 흐름 요약

```
[닉네임 입력 화면]
    ↓ (시작 버튼 클릭)
[게임 시작 → 자동차/장애물 렌더링 → 이동/충돌 감지]
           ↓ (충돌 발생)
[Game Over 화면]
          ↙ ↘
(재시작 버튼) (홈 버튼)
[게임 재시작] [메인 화면으로 이동]
```
